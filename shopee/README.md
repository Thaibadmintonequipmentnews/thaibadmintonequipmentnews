# Badminton Gear Thai

เว็บรีวิวและคู่มือเลือกอุปกรณ์แบดมินตันภาษาไทย สำหรับต่อยอดเป็นเว็บ Shopee Affiliate

## สิ่งที่มีในเวอร์ชันแรก

- หน้าแรกพร้อม hero และภาพประกอบ
- หมวดสินค้า: ไม้แบด, รองเท้า, ลูกแบด, เอ็นและกริป
- ตารางเลือกอุปกรณ์ตามสไตล์การเล่น
- ไอเดียบทความ SEO ชุดแรก
- ข้อความแจ้งเรื่องลิงก์แอฟิลิเอท

## คำสั่งใช้งาน

```bash
npm install
npm run dev
```

เปิดเว็บที่ `http://localhost:3000`

## จุดที่ต้องแก้ต่อ

- เปลี่ยนลิงก์ Shopee ปกติให้เป็นลิงก์ Shopee Affiliate ของคุณ
- เพิ่มบทความรีวิวแบบละเอียด
- ซื้อโดเมนและผูกกับ Vercel เมื่อพร้อม
