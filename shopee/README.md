# Thai Badminton Equipment News

เว็บข่าว รีวิว และคู่มือเลือกอุปกรณ์แบดมินตันภาษาไทย ภายใต้ชื่อ Thai Badminton Equipment News

## สิ่งที่มีในเวอร์ชันแรก

- หน้าแรกพร้อมบทความ Headline และบทความแนะนำ
- บทความรีวิวอุปกรณ์และข่าวนักกีฬา
- วิดีโอจากช่อง plearn
- คู่มือเลือกอุปกรณ์สำหรับมือใหม่
- ลิงก์สินค้าในบทความที่เกี่ยวข้อง

## คำสั่งใช้งาน

```bash
npm install
npm run dev
```

เปิดเว็บที่ `http://localhost:3000`

## จุดที่ต้องแก้ต่อ

- ปรับลิงก์สินค้าให้ตรงกับบทความและช่องทางที่ต้องการแนะนำ
- เพิ่มบทความรีวิวแบบละเอียด
- ซื้อโดเมนและผูกกับ Vercel เมื่อพร้อม
